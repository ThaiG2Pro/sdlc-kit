# Output Format Samples

These files are **illustrative golden samples** that agents read on demand to learn the
**shape** of each SDLC artifact (requirements, design, openapi, tasks, dev-test-report,
unit test, migration, progress). They are NOT auto-loaded into any agent's knowledge base.

The example **domain is from a reference project** (voucher/commerce) — only the
*structure* (AC-ID format, ADR layout, task breakdown, coverage table) is meant to be
reused. Replace these with samples from your own project as it matures; the agents will
keep working regardless of the example domain.
